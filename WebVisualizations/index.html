Summary: Latitude vs. X

The purpose of this project was to analyze weather changes as we get closer to the equator. Data was extracted from the OpenWeatherMap API to obtain a dataset of over 500 cities around the world.


After organizing the dataset, Matplotlib was used to plot various aspects of Weather vs. Latitude. The four factors analyzed were: Humidity, Maximum Temperature, Cloudiness and Wind Speed. This site provides the source data and visualizations created, along with a description of any trends or correlations.
